The intention of this codelab was to get you started building Android apps. We hope you want to know a lot more though, like how do I save data? How do I run background tasks? How do I display a list of photos? How do I ...

The following resources will help you learn more about Android development. They are all completely free.

# Written tutorials

- [Build your first Android App in Java](https://developer.android.com/codelabs/build-your-first-android-app#0), the tutorial this codelab was based on.
- [Android Developer Fundamentals](https://developer.android.com/courses/fundamentals-training/overview-v2)
- [Kotlin Bootcamp for Programmers](https://developer.android.com/courses/kotlin-bootcamp/overview)
- [Official Android Developer Documentation](https://developer.android.com)

# Video-based courses

- [Developing Android Apps in Kotlin](https://www.udacity.com/course/developing-android-apps-with-kotlin--ud9012)
- [Kotlin Bootcamp for Programmers](https://www.udacity.com/course/kotlin-bootcamp-for-programmers--ud9011)

We encourage you to keep learning and building. The more you build, the more you'll learn. And the more you learn, the more you'll be able to build.

> "The only way to do great work is to love what you do. If you haven't found it yet, keep looking. Don't settle." - Steve Jobs
